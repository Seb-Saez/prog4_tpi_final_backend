Comandos para levantar entorno:

(si se me vuelve a romper el entorno virtual borrarlo y no perder tiempo)
deactivate
rm -rf venv
python3 -m venv venv
source venv/bin/activate

(esto es para pruebas si estamos con el pip y el interprete python del entorno virtual venv)

which pip

python -m pip install -r requirements.txt

python -m uvicorn main:app --reload
